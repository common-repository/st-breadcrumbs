== Description ==


Simple plugin to create breadcrumb for wordpress

This section describes how to install the plugin and get it working.

1. Upload the entire st-breadcrumb-wp folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place PHP code '< ?php echo do_shortcode( '[ST-breadcrumb]' ); ? >' to display twitter feeds in your page template.
4. Place shortcode '[ST-breadcrumb]' . Use the widget to place it in your sidebar or other areas!
